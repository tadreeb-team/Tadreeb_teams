About the Internship

Aramex is looking for a motivated Computer Science student to join our Software Development team. You will work alongside experienced developers and get hands-on experience in building and maintaining web applications. This internship is ideal for students enrolled in Field Training 1.

Requirements

  • Basic knowledge of HTML, CSS, and JavaScript
  • Familiarity with any programming language (Java, Python, etc.)
  • Good communication skills
  • Ability to work in a team environment
  • University letter confirming Field Training enrollment

What You Will Learn

  • Real-world software development practices
  • Working with Git and version control
  • Agile / Scrum methodology
  • Code review and debugging skills
  • How to work in a professional development team
